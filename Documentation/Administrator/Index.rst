Installation
------------

the extention can be installed from the ter tp3_social
https://extensions.typo3.org/extension/tp3_social/

To install the extension, perform the following steps:

#. Go to the Extension Manager and install tp3_social
#. You can add tp3mods to have the advantage of further microdata
#. afterwards you should include the static template tp3_social or add the tssetup manualy

Afterwards you can use the plugin as contentelement.
for some features you mighht have to regiser a Api ID in den dependent service
